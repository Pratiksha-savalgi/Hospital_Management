# 🏥 Hospital Management System

A backend-based **Hospital Management System** built with **Spring Boot** and **MySQL**. The application provides REST APIs for managing patients, doctors, appointments, and related hospital operations. It follows a layered architecture using Spring Boot, JPA, and Hibernate.

---

## 🚀 Features

- 👨‍⚕️ Doctor Management
  - Add, update, delete, and view doctors.

- 🧑‍🤝‍🧑 Patient Management
  - Register and manage patient information.
  - Store patient details including blood group and contact information.

- 📅 Appointment Management
  - Schedule appointments between patients and doctors.
  - Retrieve appointment details.

- 🗄️ Database Integration
  - Uses MySQL for persistent storage.
  - Entity relationships implemented using JPA/Hibernate.

- ⚡ RESTful APIs
  - CRUD operations for all entities.
  - JSON-based request and response handling.

- 📝 Validation and Exception Handling
  - Input validation.
  - Custom exception handling for better API responses.

---

## 🛠️ Tech Stack

- **Backend:** Spring Boot
- **Language:** Java
- **Database:** MySQL
- **ORM:** Hibernate & Spring Data JPA
- **Build Tool:** Maven
- **API Testing:** Postman
- **IDE:** IntelliJ IDEA
